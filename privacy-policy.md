# KeySense Privacy Policy

Last updated: 2026-05-09

KeySense is an offline Android music analysis app. It analyzes audio on the user's device to estimate musical key, BPM, confidence, and related summary metrics.

## Data We Process

KeySense processes audio only while the user manually starts analysis.

Input sources:

- Microphone audio, when the user chooses microphone mode and grants microphone permission.
- Device playback audio, when the user chooses device playback mode and grants the Android screen/audio capture prompt.

## Data We Do Not Collect

KeySense does not:

- Upload audio to a server.
- Save raw microphone audio.
- Save raw device playback audio.
- Identify song title, artist, album, or account identity.
- Create user accounts.
- Use subscriptions or payment identity.

## Local History

If history is enabled, KeySense stores only analysis summaries on the device:

- Timestamp.
- Input source.
- Estimated key.
- Estimated BPM.
- Confidence.
- Analysis duration.

The user can disable history and clear history from the app settings.

## Export

When the user explicitly exports or shares results, KeySense exports only summary data. Raw audio is not included.

## Permissions

KeySense may request:

- Microphone permission for microphone analysis.
- Notification permission so Android can show the required foreground analysis notification.
- MediaProjection permission for device playback capture.

Permissions are requested only when the user starts a feature that needs them.

This policy applies to the Android application **KeySense** (`com.keysense.app`) distributed on Google Play.
